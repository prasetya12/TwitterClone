# TwitterClone
This is a simple application created using React Native and using Framework NativeBase

## Getting Started

### 1.Clone the Repo

```
$ git clone https://github.com/prasetya12/TwitterClone.git

$ cd TwitterClone

$ npm install

```

### 2.Run the Server

```
$ npm start

```

### 3. Run on Android emulator / Device

```
$ react-native run-android

```
